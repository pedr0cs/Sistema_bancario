#  Sistema Bancário Otimizado 
Desafio do bootcamp de Python da DIO. Esse projeto consiste em um sistema bancário simples.

# Tecnologias e Conceitos Aplicados
*Python

# Operações Financeiras
* Depósito: Atualiza o saldo e registra a movimentação no extrato.
* Saque: Valida o limite por saque (R$ 500,00), limite de saques diários (3 saques).
* Extrato: Recebe argumentos posicionais e nomeados. Exibe o histórico de transações e o saldo atual.
* Cadastrar Usuário (Cliente): Armazena os clientes em uma lista. Cada usuário é um dicionário contendo: Nome, Data de Nascimento, CPF e Endereço. Não é permitido duplicar CPFs.
* Criar Conta Corrente: Vincula uma conta (contendo Agência fixa "0001" e Número sequencial) a um CPF já cadastrado.
* Listar Contas: Exibe todas as contas correntes cadastradas no sistema e seus respectivos titulares.
